# 换图指南（零代码版）

你只需要 **替换 /assets 目录里的同名图片文件**（文件名不变），网站就会自动显示新图。

> ✅ 最保险做法：用你自己的新图片覆盖原文件名（保持 .jpg/.png 后缀不变）

## 1) 你要换哪些图？

| 文件名（在 /assets） | 用在什么位置 | 建议尺寸（像素） | 说明 |
|---|---|---:|---|
| **logo.png** | 顶部左上角 Logo | 400×120 | 透明底 PNG 更好 |
| **header-bg.jpg** | 顶部导航栏背景 | 2400×800 | 深色/质感图更稳重 |
| **page-hero-bg.jpg** | 所有页面的顶部大背景（Hero 背景） | 2400×1400 | 这张会影响全站第一屏氛围 |
| **home-right.jpg** | 首页右侧主视觉图 | 1600×1200 | 人物/温室/工厂都可以 |
| **platform.jpg** | Our Platform 页主图 | 1600×1200 | |
| **programs.jpg** | Programs 页主图 | 1600×1200 | |
| **supply.jpg** | Supply & Partnering 页主图 | 1600×1200 | |
| **australia.jpg** | Australia Presence 页主图 | 1600×1200 | |

## 2) 手机菜单怎么用？

- 手机宽度（≤ 860px）会自动隐藏桌面菜单，显示右侧 **三条线按钮（☰）**
- 点击按钮会展开菜单，再点任意链接会自动收起
- 点页面空白处 / 按键盘 ESC 也会收起

## 3) 换图的小技巧（避免“糊”“变形”）

- 尽量用 **JPG**（照片）/ **PNG**（Logo）
- Hero 背景图建议偏暗（或有留白），文字更好读
- 体积建议：单张图片尽量 ≤ 500KB（加载更快）
- 如果你用 Canva 导出：选择 JPG、质量 80% 左右

## 4) 我如何确认换图成功？

1. 替换 /assets 里的图片文件（同名覆盖）
2. 刷新网页（电脑：Ctrl/Cmd + Shift + R 强刷）
3. 手机上也刷新一次即可看到新图
